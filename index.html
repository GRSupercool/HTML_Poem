HTML

A simple easy to follow tutorial on Hyper Text Mark Up Language

What is HTML

HTML is an abbreviation for Hyper Text Mark Up Language. While HTML is not a programming language, HTML is required for any website, web app, and front end and web development position.

HTML 5 is the most recent version of HTML and its easy to learn. HyperText Mark Up Language is the skeleton of any website. HTML is a website's content, text, and pictures.

HTML is easy to learn, the following information is from W3Schools(w3schools.com)and Developer Mozilla(developer.mozilla.org).

Mozilla describes "HTML (HyperText Markup Language) as the building block of the web, defining the structure of web content."

"Hypertext" refers to links that connect web pages to one another, within a website or between websites.Markup is website content- words, photos, anything you see on a webpage is most likely written in HTML.

"W3Schools, describes HTML as the standard markup language for creating web pages. HTML is the standard for creating web pages, describes the structure of a web page, consist of a series of elements that tell the web browser how to display the content. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc."

The simplest way to describe HTML is the content of a website- text, links and images- labeled with element tags. Element tags describe how to write HTML using HTML syntax.

HTML Syntax

HTML syntax is a professional way of saying, HTML writing structure. HTML is written using elements, elements describe the purpose of the content and is written in tags.

In order for a web browser to display your content, images in text, you have to use a specific type of writing structure,HTML writing structure consists of elements and tags.

An element describes the content in a web page - title, body, heading, paragraph, image, articles and footers. Developers label the text and images of a web page with Elements.

Element Structure

Place the name describing webpage content between an opening and closing tag.The opening tag is the < symbol, the closing tag is the> symbol.

Writing elements is easy, simply place the purpose of the content of a webpage between opening and closing tags.
